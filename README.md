idobattan
=========

Multi platform client for [idobata.io](https://idobata.io/)

## For Developers
Be careful, This application works with only Java8 and over. 

setup is so easy. just enter the following command.

### For Linux, MacOS X, Cygwin
```bash
$ ./gradlew build
$ ./gradlew run
```
### For Windows(cmd)
```bash
$ ./gradlew.bat build
$ ./gradlew.bat run
```
